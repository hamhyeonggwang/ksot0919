## 10. 전체 신청 페이지 구성 (`register/index.html` 신규)
- **레이아웃**: 장바구니 방식 — 각 프로그램 카드를 체크박스로 선택 → 선택 항목 합산 → 한 번에 신청
- **신청 카드 목록** (7종):
  1. 보수교육 (9.19 메인데이)
  2. 구두논문 발표 (9.19)
  3. 캡스톤 디자인 (9.19)
  4. 대학원생 워크숍 (9.19)
  5. 온라인 강좌 (9.12)
  6. 오프라인 강좌 (9.13)
  7. 오프라인 강좌 (9.20)
- **UX 흐름 (내부 폼)**:
  1. 카드별 체크박스로 원하는 프로그램 선택
  2. 하단 고정 바 "선택한 N개 신청하기" 버튼 활성화
  3. 버튼 클릭 → 동일 페이지 하단 통합 신청 폼으로 스크롤
  4. 신청자 정보 1회 입력 (이름·소속·면허번호·연락처·이메일 등 — `register/ceu.html` 필드 기준)
  5. 선택한 프로그램 목록 자동 표시 (읽기 전용 요약)
  6. 제출 → 성공 메시지 표시
- **스타일**: 기존 `register/ceu.html` 폼·카드 디자인 재사용
- **기술**: 순수 HTML + Vanilla JS (빌드 도구 없음)
