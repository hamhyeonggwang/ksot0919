## 🔤 타이포그래피

| 변수 | 값 |
|------|-----|
| `--display` | `'Sora', 'Noto Sans KR', sans-serif` |
| `--body` | `'Noto Sans KR', 'Apple SD Gothic Neo', sans-serif` |

### 사용 규칙
- **영문 타이틀·숫자·라벨** → `--display` (Sora)
- **한글 본문·제목** → `--body` (Noto Sans KR)
- 한글 큰 제목은 `--body` + `font-weight:900` (Sora는 한글 미지원)

### 크기 스케일 (참고)
| 용도 | 크기 | 굵기 |
|------|------|------|
| 히어로 OT | 110px | 800 |
| 히어로 Beyond/Borders | 60px | 800 |
| 섹션 타이틀 | 40px | 800 |
| 날짜 강조 | 76px | 800 |
| 카드 제목 | 15~22px | 700 |
| 본문 | 13~15px | 400~500 |
| 캡션 | 11~12px | 400 |

## 📐 간격 · 형태

| 변수 | 값 | 용도 |
|------|-----|------|
| `--radius` | `24px` | 일반 카드 |
| `--radius-sm` | `16px` | 작은 요소 (사진, 칩 박스) |
| `--radius-lg` | `32px` | 큰 카드 (하이라이트, CTA) |

### 그림자
| 변수 | 값 |
|------|-----|
| `--shadow` | `0 4px 24px rgba(27,46,94,0.07)` |
| `--shadow-lg` | `0 20px 60px rgba(27,46,94,0.14)` |
| `--shadow-orange` | `0 10px 30px rgba(232,80,31,0.28)` |

### 섹션 패딩
- 데스크탑: `90px 0`
- 모바일(≤560px): `64px 0`

### 컨테이너
- `max-width: 1120px`, 좌우 패딩 `28px` (모바일 `20px`)

## ⚡ 애니메이션

| 이름 | 용도 |
|------|------|
| `.reveal` | 스크롤 진입 시 fade+up (JS IntersectionObserver) |
| `@keyframes float` | 히어로 글로우 오브 |
| `@keyframes spin` | AI 궤도 링 회전 |
| `@keyframes pulse` | 히어로 pill 점멸 |
| `@keyframes scrolldot` | 스크롤 힌트 |

> 인쇄 시 `.reveal`은 `print.css`에서 강제 표시 처리.
